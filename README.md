# 📚 Cuaderno de Profe · App Completa

Herramienta gratuita para docentes que permite construir un cuaderno pedagógico personalizado en PDF y gestionar notas de alumnos.

## ✨ Funcionalidades

### Constructor de PDF
- 3 portadas prediseñadas + imagen propia personalizada
- Datos del docente, calendario anual, planificación semanal
- Horario del curso, fichas de alumnos, registro de notas
- Control de asistencia, actas de reunión, notas libres y más
- Importación de alumnos desde Excel (.xlsx)

### Módulo de Notas
- 2 cuatrimestres × 4 notas por materia
- Etapa Integradora (2 o 3 notas)
- Promedios automáticos por materia y general
- Resumen final con código de colores (aprobado / en riesgo / desaprobado)
- Exportación a Excel con todas las notas y promedios

## 🚀 Uso

Accedé directamente en: **https://TU_USUARIO.github.io/cuaderno-profe**

No requiere instalación. Funciona en cualquier navegador moderno.
Todo el procesamiento ocurre localmente — los datos nunca salen de tu dispositivo.

## 🔒 Privacidad

- Sin servidor, sin base de datos
- Los datos NO se envían a ningún servidor
- Cada docente usa la app de forma completamente independiente

## 🛠️ Tecnologías

- [Alpine.js](https://alpinejs.dev/) — Reactividad
- [Tailwind CSS](https://tailwindcss.com/) + [DaisyUI](https://daisyui.com/) — Estilos
- [jsPDF](https://github.com/parallax/jsPDF) — Generación de PDF
- [SheetJS](https://sheetjs.com/) — Lectura/escritura de Excel
- [Lucide](https://lucide.dev/) — Iconografía
